### HandTalk — платформа перевода на жестовый язык

**Описание:** платформа для доступной коммуникации, включающая перевод звук/текст → жестовый язык с визуализацией (3D‑аватар) и двусторонний сценарий перевода для диалога.

**Роль и вклад:**
- Разработал backend сайта: серверная логика, API и интеграция клиентской части с сервисами перевода.
- Обучал ML‑модель для перевода жестов в текст: подготовка данных/пайплайна обучения и проверка качества на тестовых примерах.
- Поддерживал модульность решения, чтобы алгоритмы и компоненты можно было независимо обновлять и встраивать в другие продукты.

**Описание:** веб‑решение, демонстрирующее поиск маршрута для задачи коммивояжера с использованием генетического алгоритма.

**Роль и достижения:**
- Полностью реализовал генетический алгоритм (логика отбора/скрещивания/мутаций и итерационного улучшения решения).
- Разработал backend: вычисления, API/эндпоинты, связка с веб‑интерфейсом.
- Обеспечил воспроизводимость результатов и корректную обработку входных данных (точки/граф/параметры алгоритма).

### DND_AI — интеллектуальная игровая экосистема

**Описание:** система, объединяющая нарратив Dungeons & Dragons с диалоговым ИИ, способным поддерживать сверхдолгие последовательные сессии без потери контекста и личности ИИ.

**Роль и достижения:**
- Разработал каскадную систему памяти (многоуровневый стек): краткосрочный, среднесрочный, долгосрочный и “бесконечный” слои; разные подходы к компрессии и представлению (от extractive summarization до hybrid abstractive нейросетей).
- Спроектировал механики приоритизации, “иммунитета к удалению” и циклической очистки для поддержания релевантной долговременной памяти при тысячах сообщений.
- Реализовал концепцию **Infinity Memory** — долговременный когнитивный слой для хранения фактов, событий и связей, с алгоритмами приоритизации и долгосрочного удержания ключевой информации.
- Интегрировал модульную архитектуру: API-ориентированная компонента для взаимодействия диалогового движка с модулями памяти, обобщения и поиска.
